# Prerequisites
## Requirements
- IBM Cloud Account
- Option 1. Access to Watsonx Project and Watson Machine Learning
- Option 2. Access to Techzone

## Environment Setup
If you do not have existing watsonx project with Watson Machine Learning instances, please provision a techzone environment.
- Techzone: 
    - https://techzone.ibm.com/collection/tech-zone-certified-base-images/journey-watsonx
    - Reserve “watsonx.ai/.governance SaaS”
![](./Instruction-Screenshots/Techzone%20Reservation.png)

#### First time IBM Cloud users: LOOK FOR EMAIL
- If this if your first time provisioning watsonx related project, you'll receive an email from IBM Cloud to join environment, titled **Account: Action required: You are invited to join an account in IBM Cloud**
    - Click **Join now** to access IBM Cloud account

#### Watsonx Project setup
- To begin, access [watsonx](https://dataplatform.cloud.ibm.com/wx/home?context=wx) platform
- Associate Watson Machine Learning
- In project, navigate to "Manage" tab -> "Services & integrations". Click "Associate Service"  -> select a Watson Machine Learning instance.
